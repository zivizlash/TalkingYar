# TalkingYar — Discord бот с синтезом речи

Discord-бот, который превращает текстовый чат в живой поток общения! Бот автоматически превращает текст ваших сообщений в голос и воспроизводит их в реальном времени прямо в голосовом канале.
Сейчас работает в связке с небольшой оберткой qwen3-tts

---

## Установка

### 1. Развернуть restapi синта речи

https://github.com/zivizlash/talking_yar_qwen_tts_rest

### 2. Клонирование репозитория

```bash
git clone https://github.com/zivizlash/TalkingYar.git
cd TalkingYar
```

### 3. Установка зависимостей

```bash
npm install
```

### 5. Настройка окружения

Создайте файл `.env` в корне проекта с необходимыми переменными:

```env
DISCORD_TOKEN=ваш_токен_bота
```

- `DISCORD_TOKEN` — токен бота (получите через [Discord Developer Portal](https://discord.com/developers/applications))
---

## Функциональность

Бот автоматически обрабатывает сообщения, отправленные пользователями, находящимися в голосовом канале. Все ваши текстовые сообщения будут озвучены ботом через синтезатор речи

**Пример использования:**

1. Присоединитесь к голосовому каналу
2. Напишите любое сообщение (например: **"Привет всем!"**, также, с заданием голоса: **"егор: Здарова геймеры!"**)
3. Бот подключится к каналу и произнесет ваше сообщение голосом! 🎙️
