# Project Brief: TalkingYar

## Основная информация
- **Название проекта**: TalkingYar
- **Тип**: Discord бот для голосового чата с синтезом речи
- **Язык разработки**: TypeScript (Node.js)
- **Основная платформа**: Discord

## Цель проекта
Создать Discord бот который может подключаться к голосовым каналам, синтезировать текст пользователей в речь с использованием REST API, реагировать на команды и обеспечивать удобное взаимодействие в голосовых каналах.

## Текущий статус
Проект находится в активной разработке. Реализованы:
- Подключение к Discord и авторизация
- Управление голосовыми каналами
- Синтез речи через REST API (http://localhost:8000) с предварительной инициализацией голосов
- Обработка команд и сообщений пользователей
- Очередь воспроизведения с автоматической обработкой
- Персистентная очередь через Level БД

## Архитектура
Модульная структура на TypeScript с отдельными классами для разных областей ответственности:
- TalkingYarBot - основной класс бота
- Session - представление сессии для конкретного сервера
- SessionManager - менеджер всех сессий
- SpeechSynthesizer - синтез речи через REST API
- VoiceQueue - очередь сообщений

## Зависимости
- discord.js v14
- @discordjs/voice
- ffmpeg-static
- level (для персистентной очереди)
- axios (для REST API)
