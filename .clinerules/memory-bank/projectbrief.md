# Project Brief: TalkingYar

## Основная информация
- **Название проекта**: TalkingYar
- **Тип**: Discord бот для голосового чата с синтезом речи
- **Язык разработки**: JavaScript (Node.js)
- **Основная платформа**: Discord

## Цель проекта
Создать Discord бот который может подключаться к голосовым каналам, синтезировать текст пользователей в речь с использованием Gradio API, реагировать на команды голосов и обеспечивать удобное взаимодействие в голосовых каналах.

## Текущий статус
Проект находится на начальной стадии разработки. Существует базовая структура, настроены линтеры, конфигурации и основная точка входа.
