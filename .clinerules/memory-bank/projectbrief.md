# Project Brief: TalkingYar

## Основная информация
- **Название проекта**: TalkingYar
- **Тип**: Discord бот для голосового чата с синтезом речи
- **Язык разработки**: TypeScript (Node.js)
- **Основная платформа**: Discord

## Цель проекта
Создать Discord бот который может подключаться к голосовым каналам, синтезировать текст пользователей в речь с использованием Gradio API, реагировать на команды и обеспечивать удобное взаимодействие в голосовых каналах.

## Текущий статус
Проект находится в активной разработке. Реализованы:
- Подключение к Discord и авторизация
- Управление голосовыми каналами
- Синтез речи через Gradio API
- Очередь воспроизведения с использованием Level БД
- Обработка команд и сообщений пользователей
